# Homestay Review Analyzer

A responsive web application built with **Next.js** and **Tailwind CSS** to help users analyze and understand guest reviews and customer feedback for homestays.

The platform is designed to provide meaningful insights from user experiences and assist hosts in improving service quality. It offers a clean and user-friendly interface with reusable components and responsive layouts for different devices. The project aims to make review analysis easier and enhance the overall homestay experience.

## Features

* Responsive Home page with Navbar, Hero section, Cards, and Footer
* About, Dashboard, and Login pages
* Reusable UI component library
* Dark/Light mode support
* Mobile, tablet, and desktop responsive layouts
* AI feature screen for future review analysis functionality
* Clean and user-friendly interface

## Tech Stack

## Getting Started

### Clone the repository

```bash
git clone https://github.com/Niharikabutola/Homestay_Review_Analyzer
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

in your browser to view the application.
